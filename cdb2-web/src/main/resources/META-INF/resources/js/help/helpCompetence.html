Purpose

This page allows you to display the current configuration of the reference code.

How it works ?

List of instruments to be displayed

This is the list of legal instruments fow which a bailiff can be competent for. This list cannot be edited manually. It can be updated during an update of the application by your application administrator.

List of languages that can be accepted

This is a list of languqges that can be accepted by the available. This list can be edited and is automatically available in the list of accepted languages in the detail of a bailiff. To delete a language, click on the row you want to delete.
